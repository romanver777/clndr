<h3>Календарь</h3> 
<p>React, css-modules</p>

- переключение месяцев и годов, а так же выбор конкретной даты при клике</br>
- при клике по дню возвращается выбранная дата</br>
- при инициализации компонент может принимать свойство даты в формате "год-месяц-день" и переключать текущий год, месяц и день на нее. Если дата не передана, то берется текущий день</br>
- смена языка (название месяцев, дней недели)</br>
- показ дней предыдущего и следующего месяца по свойству компонента, с переходом по клику</br>
